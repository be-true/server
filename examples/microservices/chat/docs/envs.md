## <u>Logger</u>
| env        | description                                 | example | default |
|------------|---------------------------------------------|---------|---------|
| LOG_PRETTY | Отображать логи в человеко понятном формате |         | false   |
| LOG_LEVEL  | Уровень отображения логов                   |         | info    |

| env       | description      | example | default   |
|-----------|------------------|---------|-----------|
| HTTP_HOST | HOST WEB сервера |         | 127.0.0.1 |
| HTTP_PORT | Port WEB сервера |         | 3000      |
